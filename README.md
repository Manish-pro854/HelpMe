## 📌 About the Project

**HelpMe** is a modern job and work platform that connects people who need work done with those looking for work opportunities.
Users can create profiles, post jobs, and find available work through a simple and user-friendly interface.

This project showcases my skills in **React.js**, **React Router**, **Context API**, and modern frontend development using **Vite**.

---

## ✨ Features

* ✅ User Sign-Up
* ✅ Create Profile
* ✅ Post Job Functionality
* ✅ Find Work Page
* ✅ Multi-page Navigation using React Router
* ✅ Global State Management using Context API
* ✅ Responsive Navbar and Footer
* ✅ Clean and Simple UI

## 🚀 Getting Started (Run Locally)

Follow these steps to run the project on your computer.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Manish-pro854/HelpMe.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd HelpMe
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Development Server

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```
